Disintegration Effect

Kenneth Huang - CSC 471 - Fall 2018 - Christian Eckhardt

Project Description

For my final project for CSC 471 (Intro to Computer Graphics), I decided to create a disintegration effect where a model fades into dust, in a similar fashion to something that happens in a rather famous Marvel movie.

This project makes use of a geometry shader to manipulate polygons to be rendered before they go into the rasterizer and fragment shader. Since the geometry shader allows input of multiple vertices (a complete triangle, for instance), I can work with entire triangles, rather than applying calculations to only one vertex at a time.
This project also uses the model and texture of Alduin from Skyrim as a victim of this disintegration effect.

When making the geometry shader, I had three goals that I wanted to accomplish:

  • 1) Triangles should gradually float away from the model.
  • 2) Effect needs to sweep from one side to the other.
  • 3) Triangles should shrink in size while floating away.

A screenshot of dragons during decay.

I was able to address my goals by implementing the following solutions:

  • 1) Used a quadratic function to translate the triangles, using glfwGetTime() as a variable.
  • 2) Added factors based on xy-position to the time variable so that triangles on one side would float away before triangles on the other.
  • 3) Used the 3 points of the triangle to calculate an average point; used this in an weighted average with each vertex, weighted by the modified time factor mentioned in 2).

This project requires SteamVR to run, and will run on a virtual reality headset if one is connected to your computer.
